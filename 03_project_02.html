Overview

POP-UP Pirate is a toy for party and family. The rule of the game is to place the pirate into a gimmick barrel to set a lucky/unlucky slot. Inserting sword into that specific slot will launch the pirate. We transformed this game into digital environment in any mobile device.

For this project, I was in charge of implementing the majority of application including game mechanism and animation, camera image generates, database management and page routing.

Game Features

As the main part of the application, the game graphics is constructed using SVG elements. By incorporating with CSS (mainly Transform: 3d-preserve and rotateXYZ), I succeed in creating a 2.5D game environment where, despite all game graphic is constructed using 2D elements, users can rotate game elements like a 3D object. The game animation is made using CSS3. On user action, the application logic adds specific class which contains the animations.

Camera & Image Storage

The application also embeds a camera feature that players could take a picture and replace the image of the pirate face as their own faces in order provide a unique and fun user experience.

Any pictures taken by the camera and used in the gameplay are saved as persistent data in the application. I was able to store any images as a String variable by drawing the image in a HTML5 Canvas element and then converting the Canvas element into an URL using “toDataURL”. All image data is passed using Angular Service and IndexedDB.
